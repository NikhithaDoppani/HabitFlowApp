# 🎯 Habit Tracker Web Application

A modern and responsive Habit Tracker Web Application built using **HTML5, CSS3, and Vanilla JavaScript**.

The application helps users build better habits, track progress, manage daily goals, analyze performance, maintain streaks, and stay productive through an intuitive and visually appealing interface.

---

## ✨ Features

### 🔐 Authentication

* User Registration
* User Login
* Remember Me Functionality
* Forgot Password
* LocalStorage-Based Authentication

## 🚀 How to Run

1. Download or clone the repository.

3. Open `index.html` in any modern browser.

No installation required.

No npm required.

No server required.

Everything runs directly in the browser.
